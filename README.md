Coding StableDiffusion from scratch using pytorch.

## Setup environment
* Clone the repository and Go to StableDiffusion directory.
```bash
git clone https://github.com/eljandoubi/StableDiffusion.git && cd StableDiffusion
```

* Build environment.
```bash
make build
```

## Clean environment.
```bash
make clean
```
